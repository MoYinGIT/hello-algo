📚 **GESP学习路径对照**:

| GESP级别 | 对应章节 | 核心内容 |
|:---:|:---|:---|
| **L1-L2** | 第1-3章 | 编程基础、输入输出、条件循环 |
| **L3-L4** | 第4-5章 | 数组、字符串、函数 |
| **L5-L6** | 第6-8章 | 链表、栈队列、哈希表 |
| **L7-L8** | 第9-12章 | 树、图、搜索、动态规划 |
| **信奥赛** | 第13-16章 | 高级数据结构、复杂算法 |

🎯 **学习建议**: 按GESP级别循序渐进，每章配套练习题在 [Hydro平台](https://hydro.ac/d/MoYin) 同步开放。
